# TrendDetect Research Tool

A lightweight data collection tool built for academic research purposes.

## About

This project is part of a university research study on consumer behavior 
and emerging product trends across social media platforms. The tool 
collects read-only data from Pinterest to analyze keyword frequency 
and search volume patterns over time.

## Research Scope

- Platform: Pinterest (read-only access)
- Data collected: Search volume, keyword trends, save counts
- Purpose: Academic analysis only
- Duration: Limited to research period

## Privacy

This tool does not collect, store, or share any personal user data. 
All data gathered is aggregated and anonymized for research purposes only. 
No commercial use. No third-party data sharing.

## Disclaimer

This tool is developed solely for academic research. 
It is not affiliated with Pinterest in any way.
